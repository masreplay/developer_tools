# developer_tools_auto_route

Auto Route integration for [developer_tools](https://pub.dev/packages/developer_tools). Inspect current route, navigation stack, and router state from the debug overlay.

## Installation

```yaml
dependencies:
  developer_tools: ^0.0.1
  developer_tools_auto_route: ^0.0.1
  auto_route: ^11.0.0
```

## Usage

See the [developer_tools](https://pub.dev/packages/developer_tools) package for overlay setup, then use the Auto Route extension to register the tool entries.
